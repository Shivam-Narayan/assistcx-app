#### Import

Import allows documents to be added into AssistCX knowledge collections so they can be referenced by agents during task execution.

Instead of manually recreating knowledge inside the platform, existing documents can be uploaded or imported from connected repositories. This enables organizations to reuse their current documentation and make it accessible to agents as structured knowledge sources.

Imported documents become part of the selected collection and can be retrieved by agents when additional context is required.

---


#### Supported Sources

Documents can typically be imported from:

- Local files such as PDFs, DOCX, TXT, or other text-based formats  
- Connected document repositories or storage systems (if configured)

This allows teams to bring existing organizational documentation into AssistCX without recreating it manually.

---

#### After Import

Once documents are imported, they become part of the selected knowledge collection.

Agents linked to that collection can retrieve relevant content from the imported documents during task 
execution. This ensures that responses and decisions are supported by the organization’s documented 
information.
